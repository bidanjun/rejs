
## 升级到create-react-app 3.0
重点特性:通过jsconfig.json，实现绝对引用
比如我们在项目根目录创建jsconfig.json
{
  "compilerOptions": {
    "baseUrl": "src"
  },
  "include": ["src"]
}
那么对于src/components/Button.js
我们可以简单的在任何地方：import Button from 'components/Button';
